What's a Pwchi!

Welcome to Pwchie World
Pwchi is a fictive graphic character generator using HTML brock elements to convert into a single PNG image
Pwchi name is a reversed first letter from I Have Created a Wonderful PNG
Welsh language uses this word pwchi and translated to English language means please
Pwchi - character neutral / Welcome to Pwchi world
Pwchin - male / Name of this pwchin is Dan
Pwchela - female pwchi / Her pwchela name is Lorena
Pwchies - multiple characters, multiple pwchies, plural / A world of Pwchies
Pwchiman - male character / How you doing pwchiman? / What's up pwchiman?!

NAME TAB!

Text
Use input text field to Name your Pwchie or type any other short text and click the Change Name button
If your text doesn't show up, make sure to click the toggle icon on/off
Font Face changes the font used for text.
Font size, use the slider to make font bigger or smaller
Font color, click once on the color picker to change the color of the text
Add background color behind your text or use the toggle icon for on/off
Add background image behind the text. Use available options. (No Background) option means no background graphic
Toggle Opacity option to make the entire text element translucid (left for full transparency or right for full opacity)
Text position up/down using range slider
Letter Spacing option. Use the slider to add/reduce space between letters.

Pwchi App - Interface!

Main Preview
Top left window shows the Pwchi character. Click to highlight
Background options. Use the buttons to change main background image (this will not be saved)
Background change option is to allow you a better visibility of your Pwchi.

Background Environment
As a visual helper, the backgrounds option bar. Click to highlight
Background options. Use available buttons to change main background image (this will not be saved)
Background-change option is there to allow you for a better visibility of your Pwchi.

Main Buttons
Below Main Preview area there is a bar with some orange buttons. Click to highlight.
GENERATE PWCHIN - button to generate your customized pwchin ready for Save. When you're done, right-click on generated PNG to Save As...
DEFAULT - button, click if you have no idea what you done and how to remove an edit. Start fresh.
DAILY PWCHIN - button, a cool Pwchin with newspaper
REFRESH - button, refresh entire page and start over

Save and Delete
To save a Pwchin, after click on GENERATE PWCHIN button, right-click to save image as PNG in your device. Click to highlight
Your Pwchin will show-up in Generated Pwchin window below buttons. Click to highlight
To delete a generated Pwchin simply click the trash icon from rendering area. Hover over your Pwchin and you'll see a small red button with Trash icon on top of it.

Editor and Library
Pwchin App is using an intuitive interface and right section holds the Editor and Library. Click to highlight
These are the editing options and graphic elements.
